# Create SPDY and AboutCode license information file

Scripts to create the lib/license/license_information.js file. This file is used to store the license information from
spdx and aboutcode.

Execute the scripts 'aboutcode-importLicenses-git.js' and 'spdx-importLicenses-git.js' to import the license
information from aboutcode and spdx respectively.

Copy the results into the 'lib/license/license_information.js' file.
